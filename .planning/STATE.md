# Project State: Newman Yange Portfolio

## 📍 Current Status
The project is a static portfolio website for **Newman Yange** (Rapper, FX Trader, Founder of NYNTH World). It is currently a single-page application with a modern, high-fidelity aesthetic using Tailwind CSS and custom animations.

## 🏗️ Tech Stack
- **Frontend**: HTML5, Tailwind CSS (CDN), custom CSS.
- **Typography**: Montserrat (Body), Playfair Display (Headings).
- **Icons**: Font Awesome 6.
- **Assets**: Local images (chairs, meetings, rapper photos).

## 🧩 Core Components
- **Loader**: Animated particle loader with percentage tracker.
- **Navigation**: Sticky transparent header with mobile menu.
- **Hero**: Full-screen background with call-to-action.
- **About/Early Life**: Biographical sections with grid layouts.
- **NYNTH World**: Brand section featuring a custom gallery/carousel.
- **Music**: Cards linking to Spotify, Apple Music, and Audiomack.
- **Contact/Footer**: Social links and copyright.
